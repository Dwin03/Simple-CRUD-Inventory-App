# Simple CRUD Inventory 📱

A basic inventory management system that allows users to Create, Read, Update, and Delete (CRUD) product records. It uses a local database to store and manage inventory data efficiently.

---

## 🚀 Features

* Add new inventory items (Create)
* View all stored items (Read)
* Update existing item details (Update)
* Delete items from the inventory (Delete)

---

## 🧠 How It Works

1. User selects an operation (Add, View, Update, Delete).
2. The system stores and retrieves data using a local database.
3. Changes are reflected instantly in the inventory list.

---

## 🛠️ Tech Stack

* Kotlin - Backend
* Room Database - Local database
* Android Studio - Coding Platform
